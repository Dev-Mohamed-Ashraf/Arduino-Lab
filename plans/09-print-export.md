# 09 — Print / Export

## الهدف
ورقة الحجز الرسمية — قابلة للطباعة أو الحفظ PDF.

## القرار التقني

**طباعة من المتصفح، مش توليد PDF على السيرفر.**

| الطريقة | الحكم |
|---|---|
| Puppeteer على Render | ❌ Chromium ~300MB — بياكل الفري تير كله |
| `@react-pdf/renderer` | ❌ تشكيل الحروف العربية بيتكسر (الحروف بتتفصل ومعكوسة) |
| `pdfmake` / `jsPDF` | ❌ نفس مشكلة العربي، ومحتاج تضمين خطوط يدوي |
| **صفحة HTML + `@media print`** | ✅ العربي بيترندر بمحرك المتصفح = مضبوط 100%، تكلفة سيرفر صفر، والمستخدم يحفظ PDF من نفس ديالوج الطباعة |

## المخرجات

- `apps/student/app/booking/[bookingNumber]/print/page.tsx` — Server Component
- `apps/student/app/booking/[bookingNumber]/print/layout.tsx` — layout نضيف بدون sidebar/nav
- `apps/student/app/booking/[bookingNumber]/print/print.css`
- نفس الصفحة متاحة من لوحة الأدمن (رابط لدومين الطلبة)

## قواعد الـ print CSS

```css
@page { size: A4; margin: 12mm; }

@media print {
  .no-print { display: none !important; }
  body { background: #fff; color: #000; font-size: 11pt; }
  a[href]::after { content: none; }          /* ما يطبعش الروابط */
  table { page-break-inside: avoid; }
  tr, .avoid-break { page-break-inside: avoid; }
  thead { display: table-header-group; }      /* رأس الجدول يتكرر لو اتقسم صفحتين */
  .page-break { page-break-before: always; }
}
```

- ألوان الخلفية ما بتطبعش افتراضياً → الحدود والخطوط بالأسود، مش اعتماد على خلفيات ملوّنة.
- الخط العربي متضمّن عبر `next/font` — بيشتغل أوفلاين وفي الطباعة.
- زرار «طباعة» (`.no-print`) بينادي `window.print()`.
- الصفحة تتفتح كاملة على الشاشة كمان (مش للطباعة بس).

## التحقق
- فتح `/booking/ARD-2026-0001/print` → الورقة ظاهرة كاملة
- `Ctrl+P` → المعاينة صفحة A4 واحدة، العربي سليم متصل، مفيش نص مقطوع
- حفظ PDF من ديالوج الطباعة → الملف نضيف والنص قابل للتحديد (مش صورة)
- حجز بـ 15 مكون → الجدول بيتقسم صح والرأس بيتكرر
- الطباعة من موبايل شغّالة
